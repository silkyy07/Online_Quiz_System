🧩 Project Overview

This project is a Web-Based Quiz Application that allows users to attempt quizzes and receive instant results. It is designed to make the examination process simple, fast, and interactive without requiring any backend or database setup.

Users can enter their name, select quiz level and timer, attempt randomly generated questions, and get immediate feedback with their score.

---

🎯 Features

- 👤 User enters name before starting quiz
- ⏱️ Selectable timer for quiz duration
- 📊 Three difficulty levels: Easy, Medium, Hard
- 🔀 Random selection of questions
- 🧠 Automatic answer evaluation
- 📈 Instant score and performance feedback
- 💾 High score stored using Local Storage
- 🔁 Option to retry quiz
- 💬 Motivational messages based on performance

---

🛠️ Technologies Used

- HTML – Structure of the application
- CSS – Styling and layout
- JavaScript – Logic, timer, validation, scoring
- Local Storage – Client-side data storage

---

🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open "Quiz.html" in any web browser
4. Start the quiz and explore the features

---

🔄 Project Flow

1. User enters name and selects level & timer
2. Clicks on Start Quiz
3. Questions are displayed randomly
4. Timer starts countdown
5. User submits answers
6. System calculates score
7. Results and correct answers are displayed
8. High score is stored

---

🌟 Advantages

- User-friendly interface
